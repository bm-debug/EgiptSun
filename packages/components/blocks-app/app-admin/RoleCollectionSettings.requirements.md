# Требования к компоненту настроек кабинета/коллекций

Ниже — требования к UI-компоненту настроек, сформулированные «человеческим» языком.

## Элементы интерфейса

> Описываем только то, что пользователь видит/настраивает в UI.

### Выбор контекста

- **Роль/кабинет**: селект роли, для которой редактируются настройки (роль `Administrator` не редактируется).

### Раздел «Кабинет»

- **Общий title кабинета (мультиязычный)**: поле(я) для редактирования названия кабинета по всем языкам из настроек проекта.
- **Base URL**: поле для базового URL кабинета (абсолютный URL).
- **Redirect URL после входа**: поле «куда перенаправлять пользователя после успешного логина».

### Раздел «Коллекции»

- **Список коллекций**: интерфейс добавления/удаления коллекций (какие коллекции доступны в кабинете).
- **Настройки выбранной коллекции** (для каждой коллекции из списка):
  - **Title коллекции (мультиязычный)**: редактирование названия коллекции по всем языкам из настроек проекта.
  - **CRUD‑действия**: переключатели доступных действий с коллекцией:
    - `Read` **всегда включён по умолчанию** (и не должен выключаться).
    - `Create`, `Update`, `Delete` — включаются/выключаются отдельно.
  - **Колонки таблицы**:
    - список **доступных** колонок (приходит из данных коллекции);
    - список **выбранных** колонок для отображения;
    - возможность менять порядок выбранных колонок;
    - возможность скрывать/показывать колонку.

### Раздел «OLAP»

- **Настройка OLAP для коллекции**: для каждой коллекции (где это уместно) должна быть возможность включить и настроить OLAP‑вкладки.
- **Список OLAP‑вкладок** (настраивается массивом вкладок, соответствует типам из `BaseCollection.ts`):
  - поля вкладки:
    - `id`
    - `collection`
    - `localKey`
    - `foreignKey`
    - `label` (**мультиязычный**, редактируется по всем языкам из настроек проекта)
  - действия в UI:
    - добавить вкладку
    - удалить вкладку
    - изменить порядок вкладок

### Сохранение и обратная связь

- **Кнопка “Сохранить”** и состояния: загрузка/сохранение/ошибка.
